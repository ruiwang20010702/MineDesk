   - 检查端口: `lsof -i :17860`
   - 查看日志: `tail -100 /tmp/minecontext.log`

2. LLM 调用失败
   - 验证配置: `curl http://127.0.0.1:17860/api/model_settings/get`
   - 检查网络: `ping api.siliconflow.cn`
